# Backup fix (hooks / UI / metadata)

In questa cartella vanno **solo** le copie di sicurezza dei singoli file modificati durante una fix
(nome tipo `backup-*-stabile.php` / `.js` / `.json`).

**Non** inserire qui:

- export completi della cartella `custom/` (usare `exports/custom/` e `php tools/export-custom-for-github.php`)
- ZIP generici o dump dell’intero CRM
